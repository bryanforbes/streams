constructor

  • new SeekableStream(underlyingSource: Source<T>, strategy?: Strategy<T>, preventClose?: boolean): SeekableStream

  • Overrides ReadableStream.constructor

    • Defined in SeekableStream.ts:9

    Parameters

    • underlyingSource: Source<T>
    • Default value strategy: Strategy<T> = {}
    • Default value preventClose: boolean = true

      (default=true) Prevent the stream from closing when it reaches the end. If true, the stream will not close when requestClose is called on the controller (which is typically done by the source when it reaches its end). This allows for re-seeking in a stream that has already been read to the end. The stream can be closed by calling ReadableStream#close.

    Returns SeekableStream
