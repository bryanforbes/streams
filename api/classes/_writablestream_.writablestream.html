Class WritableStream<T>

This class provides a writable stream implementation. Data written to a stream will be passed on to the underlying sink (WritableStream.Sink), an instance of which must be supplied to the stream upon instantation. This class provides the standard stream API, while implementations of the Sink API allow the data to be written to various persistence layers.
