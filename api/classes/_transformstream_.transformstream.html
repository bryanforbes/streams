Class TransformStream<R, W>

A TransformStream is both readable and writable. Its purpose is to apply some transform logic to everything that is written to it and provide the transformed data via its reader. As such, it requires no ReadableStream, WritableStream, or Source or Sink to be supplied - it provides its own.

It does require an object that implements the Transform interface to be supplied. The transform method will be applied to all data written to the stream.

The readable stream API is available via the TransformStream's readable property, which is a ReadableStream. The writable stream API is available via the TransformStream's writable property, which is a WritableStream.
